Docker
  Namespace ......... Isola processos
  Cgroups ........... Controla recursos
  OverlayFileSystem . Sobreposição de imagens

  Commands:
      - volumes |> bind/path:
                    Use mounts for external volumes
                    docker run --name nginx -d --mount type=bind,source="$(pwd)":/usr/share/nginx/html nginx
                |> create
                    docker volume create VOLUME_NAME
                |> mount/volume
                    docker run --name nginx -d --mount type=volume,source=my_volume,target=/usr/share/nginx/html nginx

  Build:
    docker build -t IMAGE_NAME .

  Dockerfile:
    ENTRYPOINT - Usado para iniciar comandos e não é afetado pelos argumentos do docker run
    CMD - Paramêmtros do ENTRYPOINT é afetado (sobrescrito) pelos argumentos do docker run

  Network
    Bridge: Permite a comunicação entre os containers que estão na mesma rede
    Host: Mescla a network do docker com a rede do host
    Overlay: Comunicação de rede em máquinas externas (fisicamente)
    Maclan: Atribui um endereço MAC à interface de rede virtual de cada contêiner, fazendo com que pareça ser uma interface de rede física diretamente conectada à rede física.
    None: Isola o container
